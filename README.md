# SistemaEscolar
simulação de um sistema escolar, para por em prática meus conhecimentos

Nele haverá o cadastro de professores e alunos, cada aluno terá seu cadastro com código de identificação, nome, email, nota;
O professor poderá lançar notas e situação de aprovado, retido ou reprovado;
cada professor terá sua matéria, o professor terá acesso a lista de alunos apenas da matéria dele;
haverá uma lista geral de alunos e professores, entretanto não será acessado por qualquer pessoa;
haverá também cadastro de pessoa, para que cada aluno acesse apenas suas devidas informações
